Hecatonchire 0.1.0 (2012-02-06)

This is the first official branch with all features to date included.

  • Remote page memory access at a setup per process
  • Shared virtual address space with transparent coherency mechanisms
  • RDMA support: softIWARP; IWARP; RoCE; InfiniBand
  • Remote memory redundancy via memory provider mirroring
  • Remote page simple pre-fetcher
  • Native kernel API (via ioctls and sysfs)
  • User mode binding via libheca library
  • Qemu providing transparent (via Linux/KVM) memory aggregation
  • Qemu providing post-copy live migration (via Linux/KVM)
