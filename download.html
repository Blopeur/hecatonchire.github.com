Downloads

All of the code you will need to run the heca platform is available on github. We advise reading the getting started guide before starting.

View code here

To install the heca kernel, you will need to first download the linux-heca repository from github. This can be done by cloning the source with git using this command:

git clone http://github.com/hecatonchire/linux-heca.git

The libheca module is available from the libheca repository from github. This can be done by cloning the source with git using this command:

git clone http://github.com/hecatonchire/libheca.git

The qemu-heca code for doing live migration is available from the qemu-heca repository from github. This can be done by cloning the source with git using this command:

git clone http://github.com/hecatonchire/qemu-heca.git
