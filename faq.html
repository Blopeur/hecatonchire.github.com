Frequently Asked Questions

What license is the hecatonchire project under?

Apache License


What languages do you support?

The libheca API is written in C so supports C/C++


Is it possible to use libheca without installing the heca kernel?

No, you will need to install the heca kernel.


How can I contribute to the hecatonchire project?

Become a member of the team on github. All contributions are welcome.
