The Vision

Cloud storage has become pretty popular recently allowing storage to become a service product and therefore virtually limitless. Imagine being able to aggregate memory, cpu and I/O in exactly the same way. Imagine servers being able to request additional memory or computing power from other networked nodes on demand. This is what the hecatonchire project aims to deliver.
