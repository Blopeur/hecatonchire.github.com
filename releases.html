Heca Release Notes

Heca-v0.1.0 (06 February 2012)

This is the first official branch with all features to date included.

heca-linux features

  • Full Integration with Linux MM
  • Shared Virtual Address Space
  • RDMA and zero-copy
  • Page pool
  • RRAIM (active-passive-passive)
  • Page prefetching
  • Coherent sharing

libheca features

  • Simple API for master and client initialization

qemu-heca features

  • Transparent memory aggregation
  • Live migration
